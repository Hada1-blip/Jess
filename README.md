# Conciergerie by Jess — Site Web

Site professionnel pour **Conciergerie by Jess**, conciergerie privée en Île-de-France.

## Fichiers

- `index.html` — Accueil
- `services.html` — Services
- `galerie.html` — Galerie
- `reservation.html` — Demande de service
- `contact.html` — Contact
- `style.css` — Styles partagés
- `script.js` — JavaScript partagé

## Déploiement GitHub Pages

1. Créez un dépôt GitHub
2. Uploadez tous les fichiers à la racine
3. Settings → Pages → Deploy from branch main / (root)
4. Le site sera accessible sous : https://votre-username.github.io/nom-depot/
